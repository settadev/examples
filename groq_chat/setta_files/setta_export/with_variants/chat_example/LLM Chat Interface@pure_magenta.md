# LLM Chat Interface

## How to use

1. Create a secrets.json file in the folder that you started Setta in.

2. Add the following:
```json
{"api_key": <your api key>}
```
where <your api key> is replaced by your actual Groq api key. 

3. The `Global Variables` section should now have your `api_key`.

4. Click Run in the nav bar.

5. Start chatting with llama3.
