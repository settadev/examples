# AutoML on tabular data using SapientML

## Install packages

Activate your python environment and then run this command:
```python
pip install -r requirements.txt
```

## Start Setta

Run this in your terminal:

```
setta
```

Then in your browser go to http://127.0.0.1:8000/.
